\begin{zusammenfassung}
\lang{de}{
Wir betrachten in diesem Kapitel zunächst die verschiedenen Darstellungsweisen von Geraden und Ebenen im $n$-dimensionalen Raum.
\\\\
Die Parameterform beschreibt Geraden anschaulich durch einen Punkt und einen Richtungsvektor. Wir werden sehen, wie äquivalent dazu auch eine lineare Gleichung dargestellt werden kann, die wir als Koordinatenform bezeichnen. Diese Konzepte wenden wir dann auch auf Ebenen an.
\\\\
Schließlich analysieren wir die Lagebeziehungen und Abstände zwischen Punkten, Geraden und Ebenen im dreidimensionalen Raum.
}
\lang{en}{
In this chapter we consider first the different ways in which lines and planes in $n$-dimensional 
space can be represented.
\\\\
The parametrised form of a line uses a single point on the line and the direction vector of the 
line to describe it. We see in this section how these can also be used to represent the line with 
conditions on coordinates rather than with vectors. These concepts are then applied to planes.
\\\\
Finally, we cover the various types of intersections and relationships between points, lines and 
planes in $3$-dimensional space, and how we can find the distance between them in each of these cases.
}
\end{zusammenfassung}
